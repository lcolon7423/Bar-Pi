Instructions- Making a gif

Gifs are use because I wanted a transparent background.

In order to make my gifs, I used paintshop pro 5 and Animataion Shop
If these links don't work, then do a google search.. If you are unable to find these programs, you can use any other paint program that resizes or any animated gif maker that has a transparent background.

Before you create a drink you have to add the liqour or mixer to the drink or Shot library if you don't know how click here
Add a drink

Once the new ingredient in entered it will make a generic pic.

Go online and find the image of the ingredient you just added and save it to your harddrive

Open image in paint shop pro

use the selection tool to crop image to its smallest size then copy and patse to create new image

Select image and resize..

Set to pixel size and change the height to 275 make sure maintain aspect ratio is selected then click ok

Save image as the name of the ingredient

Open new image in animation pro shop

Select flood fill and check the box To Canvas Color

click on background

Save image as gif in the same name

From the nav bar goto settings then select link image

There you will see all the ingredients that don't have an image attached

Click browse and select the image you just created then click update image

To check image see stats
