Add a drink

Description: To add a new drink to the database

Under Settings in the nav bar click add drink

Enter the name of the drink (Captialize first letter to be sure it's sorted propery)

If the drink isn't in the database you will be brought to add drink form

From the Dropdown menu add the ingredients if the ingredient isn't there, select other then type in the name in Enter new Ingredient

Enter the amount

If you have more ingredients to add, select addmore then click add

When you about to add the last ingredient, do not select addmore and click add

You drink as been added to the library

If you add the wrong ingredient continue making the drink then update drink to fix it

Note: You can only make drinks that have the SAME NUMBER OF PUMPS as you do ingredients if you have only 3 pumps you will only be able to make drinks with 3 ingredients. You can make a drink having more but you will not be able to make it and it will not show up on your drink list

Update Drinks

In update select update drinks

Choose the drink you wish to change

The drink will list all of the ingredients and quantity

Here you and click add to add an ingredient

when you add to the drink you will have to select update to add the ingredient

Click delete to delete the drink

To change each ingredient click on the ingredient

Clicking on the ingredient will give you 2 more choices Update or delete

Update will give you the option to change the amount

Delete will remove that ingredient

To exit click home or Bar-Pi on the nav bar

My Liquor Cabinet

Create my liquor cabinet

From either settings or Pumps click Load Pump

Choose any one of the existing liqour or mixers to load

Choose to load Add My Cabinet

Select choose who's cabinet you want to put into

If the name doesn't exist set to other then add name to New Owner

check addmore if you wish to add to my cabinet

Access my liquor cabinet

from Drinks click on my liquor cabinet

Select the name from the list

On the bottom you can choose from drinks or shots

Remove from my liquor cabinet

Inside of my liquor cabinet select the liquor or mixer you want to remove

Click on the image to remove from cabinet

Add Pump

In pumps click add Pump

Listed will be the next pump

choose the pin you wish it to be on

choose the flow rate(how many seconds it takes to pour an ounce)

click add Pump

A empty bottle will be set to it, load whatever you want to it

Note: pin 40 is reserved for trigger switch used if you or whoever is making a drink forgets to put a glass

Back to Top




Update Pump

In pump click Update Pump

You will have a list of the pins on each of the pumps and flow rate

Pick the pump you wish to change then click enter

Change either pump pin or the flow rate or both then hit enter

Test Flow

Place measuring cup under spout

Start off with one second

Bar-Pi will run the pump for one second.

If the liquid is not an ounce add more time until it becomes an ounce

Click Save

If you go over, Click reset and try again

Back to Top
